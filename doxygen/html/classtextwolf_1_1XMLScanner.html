Detailed Description

template<class InputIterator, class InputCharSet_ = charset::UTF8, class OutputCharSet_ = charset::UTF8, class OutputBuffer_ = Buffer, class EntityMap_ = std::map<const char*,UChar>>
class textwolf::XMLScanner< InputIterator, InputCharSet_, OutputCharSet_, OutputBuffer_, EntityMap_ >

XML scanner template that adds the functionality to the statemachine base definition.

Template Parameters:
InputIterator input iterator with ++ and read only * returning 0 als last character of the input
InputCharSet_ character set encoding of the input, read as stream of bytes
OutputCharSet_ character set encoding of the output, printed as string of the item type of the character set,
OutputBuffer_ buffer for output with STL back insertion sequence interface (e.g. std::string,std::vector<char>,textwolf::Buffer)
EntityMap_ STL like map from ASCII const char* to UChar
